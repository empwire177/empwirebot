# Contributing to EmpwireBot

Thank you for your interest in EmpwireBot.

EmpwireBot is a personal robotics project focused on Raspberry Pi 5, ESP32, ROS 2, sensors, LiDAR, camera, mapping, and autonomous navigation.

At this stage, the project is mainly maintained by the project owner, but contributions, ideas, testing notes, and improvements may be welcome in the future.

---

## Project Goals

EmpwireBot is being developed step by step with the following goals:

* Stable robot hardware
* Clean ESP32 firmware
* ROS 2 integration
* Sensor monitoring
* Encoder-based odometry
* LiDAR mapping
* Camera vision
* Autonomous navigation
* Clear documentation

---

Possible future contribution areas:

* Camera testing
* Documentation fixes
* Wiring improvements
* 3D printed parts
* Navigation and SLAM experiments

---

## Contribution Guidelines

Please follow these basic rules:

1. Keep changes clear and simple.
2. Test code before submitting it.
3. Document wiring or pin changes.
4. Do not commit build files, temporary files, or large media files.
5. Use clear commit messages.
6. Keep EmpwireBot documentation beginner-friendly.

---

## Commit Message Style

Good commit message examples:

```text
Add encoder test firmware
Update ESP32 pin map
Fix sensor serial parsing
Add LiDAR setup notes
Improve ROS 2 workspace documentation
```

## Project Structure

Main folders:

```text
docs/         Project documentation
firmware/    ESP32 firmware
hardware/    Wiring, pin maps, and hardware notes
images/      Project images and diagrams
ros2_ws/     ROS 2 workspace and packages
```
